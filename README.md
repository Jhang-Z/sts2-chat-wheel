## 1. 安装 Mod

### Windows

**第一步：找到 mods 文件夹**

```
C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2\mods\
```

> 找不到？在 Steam 库中右键游戏 → **管理** → **浏览本地文件**，然后手动建一个 `mods` 文件夹。

**第二步：下载并解压**

1. 前往本项目的 [Releases](../../releases) 页面
2. 下载 `sts2_chat_wheel-0.2.0.zip`
3. 右键 zip → **全部解压**，得到一个 `sts2_chat_wheel` 文件夹
4. 把整个 `sts2_chat_wheel` 文件夹拖进 `mods\` 目录

---

### macOS

**第一步：开启 Rosetta（Apple Silicon Mac 必须）**

1. Finder → **应用程序 → Steam**，找到游戏；或在 Steam 中右键游戏 → **管理** → **浏览本地文件**
2. 找到 `SlayTheSpire2.app`，右键 → **显示简介**
3. 勾选 **使用 Rosetta 打开**

> ⚠️ 跳过这步游戏和 Mod 可能无法正常运行。

**第二步：找到 mods 文件夹**

`⌘ Shift G`，粘贴：

```
~/Library/Application Support/Steam/steamapps/common/Slay the Spire 2/SlayTheSpire2.app/Contents/MacOS/mods/
```

**第三步：下载并解压**

下载 `sts2_chat_wheel-0.2.0.zip`，双击解压，把整个 `sts2_chat_wheel` 文件夹拖进 `mods/` 目录。

---

## 2. 启用 Mod

1. 启动游戏
2. 主界面 → **Mod 菜单** → 启用 **StS2 Chat Wheel**
3. 重启游戏

> ⚠️ **联机时所有玩家都需要装相同版本**，否则消息握手失败，SL / 状态播报这类功能不会跨人生效。

## 3. 配置豆包 TTS（可选）

不配置 Key 也能用——Mod 会以纯文字气泡模式运行，队友看得到字但听不到语音。想要语音：

### 第一步：注册火山引擎账号

[volcengine.com](https://www.volcengine.com/) 注册并完成实名认证。

### 第二步：开通豆包 TTS

1. 控制台 → **智能语音 → 语音合成**
2. 开通 **豆包语音合成模型 2.0（Doubao TTS 2.0）**
3. **音色库** 里给想用的音色点 **授权**（推荐先授权「爽快思思·女声」）

> 火山引擎有免费额度（约 20,000 字符），日常测试够用。

### 第三步：拿 API Key

控制台 → **账号 → 密钥管理** → **新建 API Key**，复制生成的 Key（`xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`）。

### 第四步：填进游戏

1. 进游戏，按 `;` 打开设置页
2. **设置** 标签 → **豆包 TTS API 密钥** → 粘贴 Key
3. **测试连接** → 看到 `✓ 连接成功` → **保存**

下次进游戏即生效。

> **第一次说某句话**会调豆包 API（~500ms），合成后缓存到本地；同样的「文字 + 音色 + 情感」之后都是本地秒播。

## 4. 功能一览

### 4.1 聊天轮盘

| 操作 | 效果 |
|---|---|
| 按住 `Y` | 弹出 16 扇区轮盘（**内圈 8** 闲聊 / **外圈 8** 战术） |
| 按住 `Y` 移动鼠标 | 内/外圈按距离自动切换；高亮显示完整文本 |
| 松开 `Y` | 发送词条 + 播语音；队友同时看到气泡 + 听到声音 |
| 按住 `Y` 时按 `Esc` | 取消，不发送 |

### 4.2 状态一键播报

按住 **`Cmd`（Mac）/ `Ctrl`（Win）+ 鼠标右键** 点击下面任何对象，自动广播给全队：

| 点哪 | 文案 |
|---|---|
| 敌人 | "都打这一只(血量 30/50)" + 红色聚焦箭头 + 音效 |
| 药水 | "我有【火焰药水】" |
| 增益/减益图标 | "我处于【力量2】的状态" |
| 遗物 | "我有遗物【燃烧之血】" |
| 顶栏金币 | "我有 150 金币" |
| 顶栏血量 | "我血量 75/80, 格挡 5"（格挡 > 0 才追加） |
| 顶栏牌组按钮 | "我牌组 32 张" |
| 战斗能量球 | "我能量 3/3" |
| 抽牌堆 / 弃牌堆 / 消耗堆 按钮 | "抽牌堆 8 张" / "弃牌堆 3 张" / "消耗堆 2 张" |

### 4.3 快速 SL（存档读档）

按 **`Cmd Shift R`（Mac）/ `Ctrl Shift R`（Win）**：

- **单人**：立刻断开 → 自动重连 → 直接回到这场战斗之前的房间。
- **多人**：弹出投票框，所有玩家点「立即执行 SL」全票通过 → 全员同时退出 → 房主自动重开 lobby → 队友自动重连 → 自动 ready → 游戏自动继续。任何人点「取消」取消SL。

> SL 的回滚点是**当前房间开始之前**（游戏自带的房间结束 autosave）。回合内 SL 不支持。
>
> 多人 SL **仅 Steam 联机** 自动重连。直连 IP（ENet）模式只会自动断开，需要手动重连。

### 4.4 自动提示

不需要任何操作，开战自动播报：

- **手牌分析**：每回合开始扫一遍手牌，如果有可打出（能量够）的「给予易伤 / 给予虚弱」类牌，自动播报"我有【易伤】牌"。条件牌（如"每层易伤造成额外伤害"）不触发误报。
- **威胁预警**：敌方意图扫描，如果当前血量 + 格挡扛不住敌方下回合的总伤害，自动播报"危险！我可能被秒"。每回合最多一次，不刷屏。

## 5. 设置页说明

按 **`;`** 打开设置，三个标签页：

### 轮盘配置

左侧 16 槽位轮盘预览（内圈 8 / 外圈 8）。点击某槽位后：

- 下方 **编辑** 区改文字（最多 20 字）
- 开关 **语音** 并选择情感（正常 / 开心 / 愤怒 / 无奈 / 委屈 / 嘲讽 / 惊讶 等共 14 种）
- 点 **▶ 试听** 预览

右侧是 **语音库**，36+ 条预设词条。点击直接写入当前选中槽位。

### 设置

- **快捷键**：点击按钮 → 按一个键重绑（轮盘 / 设置 / SL 三个键独立可改）
- **音色**：91 个豆包 TTS 2.0 音色任选（爽快思思 / 猴哥 / 海绵宝宝 / 灰太狼 / ...）
- **豆包 TTS API 密钥**：填 Key 并测试连接

改完点 **保存** 立即生效，无需重启。

## 6. 常见问题

**Q：按 `Y` 没反应？**  
A：Mod 是否在游戏 Mod 菜单里启用？是否在对局中（大厅界面轮盘不弹出）。

**Q：macOS 崩溃 / Mod 不加载？**  
A：检查是否勾了 **使用 Rosetta 打开**（第 1 节 macOS 部分）。

**Q：有语音但队友听不到？**  
A：语音本地播放，气泡才广播给队友。如果队友连气泡都看不到，检查双方是否都装了 Mod 且**版本一致**。

**Q：测试连接报 WebSocket 错误？**  
A：通常是 API Key 无效或没开通对应服务。回火山引擎控制台确认 Key 状态、TTS 2.0 是否已开通并授权了你选的音色。

**Q：第一次发的语音有 0.5 秒延迟？**  
A：第一次合成某句话才需要调 API，之后同一组合从本地缓存播放，零延迟。

**Q：多人 SL 队友卡在主菜单 / 看到「连接超时」？**  
A：双方都更新到最新版本（Steam 模式有自动重连，旧版本不支持）。如果偶尔还是卡，按一下 **多人模式 → 加入 → 选房主** 手动重连即可。

**Q：自动提示太烦怎么关？**  
A：当前版本暂不支持单独关闭手牌/威胁分析。下个版本加。

**Q：SL 后金币、HP、Boss 记录会异常吗？**  
A：不会。SL 用的是游戏自带 autosave + 自带载入流程，对游戏来说就是「退出再继续」。`SerializableRun.NumReloads` 字段会 +1（仅排行榜会用，对正常游戏无影响）。**Daily Run 不要 SL**——分数会失效。
